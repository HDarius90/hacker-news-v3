# Hacker News V3.0

A modern, minimal Hacker News redesign built with **React**, **TypeScript**, and **Tailwind CSS**, using the official [Hacker News API](https://github.com/HackerNews/API).

## 🚀 Tech Stack

- React + TypeScript (Vite)
- Tailwind CSS
- Cypres + Jest

## ⚙️ Setup

```bash
npm install
npm run dev
```
